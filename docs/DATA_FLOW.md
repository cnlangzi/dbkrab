### 3.2 Snapshot 分页查询

```sql
-- 第一次查询：使用 TOP + ORDER BY
SET ROWCOUNT 10000;
SELECT * FROM dbo.orders ORDER BY order_id;
SET ROWCOUNT 0;

-- 后续查询：使用 ROW_NUMBER() 分页
SELECT * FROM (
    SELECT *, ROW_NUMBER() OVER(ORDER BY order_id) AS _rn
    FROM dbo.orders
) AS _page
WHERE _rn > 10000 AND _rn <= 20000;
```

│                                                                                 │
│  LSN 推进规则:                                                                   │
│                                                                                 │
│  1. Poller 获取 changes 后:                                                     │
│     - last_lsn = 最后一条 change 的 LSN                                          │

│                                                                                 │

│                                                                                 │
│  3. Snapshot 完成后:                                                            │
