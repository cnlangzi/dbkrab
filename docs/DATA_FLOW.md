### 2.2 GetChanges 详细流程

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│                        cdc.Querier.GetChanges()                                 │
├─────────────────────────────────────────────────────────────────────────────────┤
│                                                                                 │
│  输入参数:                                                                        │
│    - captureInstance: CDC 捕获实例名 (如 "dbo_orders")                            │
│    - fromLSN: 起始 LSN                                                           │
│    - toLSN: 结束 LSN (nil 表示到最大 LSN)                                          │
│                                                                                 │
│  SQL 调用:                                                                        │
│    SELECT * FROM fn_cdc_get_net_changes_<capture_instance>                       │
│    (@from_lsn, @to_lsn, 'all')                                                 │
│                                                                                 │
│  返回字段:                                                                        │
│    - __$start_lsn     → LSN                                                     │
│    - __$operation     → Operation (1=DELETE, 2=INSERT, 3=UPDATE_BEFORE, 4=UPDATE_AFTER) │
│    - __$transaction_id → TransactionID                                           │
│    - __$commit_time   → CommitTime                                              │
│    - 数据列           → Data (map[string]interface{})                            │
│                                                                                 │
└─────────────────────────────────────────────────────────────────────────────────┘
```

### 6.3 操作类型映射

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│                           操作类型映射                                           │
├─────────────────────────────────────────────────────────────────────────────────┤
│                                                                                 │
│  CDC Operation          →    Sink Type    →    说明                             │
│  ───────────────────────────────────────────────────────────────────────────   │
│                                                                                 │
│  OpInsert (2)          →    SinkUpsert    →    插入或更新 (根据 PK)              │
```
